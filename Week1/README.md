# SoundX 🎧

SoundX is a simple headphone landing page project made using HTML, CSS, and JavaScript. The website is designed to showcase different headphone products with a clean modern UI and smooth navigation.

This project was created for practice and learning frontend web development concepts like layout designing, responsive sections, navigation handling, and styling.

---

## Features

- Modern navigation bar
- Smooth scrolling navigation buttons
- Product cards section
- Features section
- Contact form
- Responsive flexbox layout
- Clean and attractive UI design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Project Files

- `index.html` → Main webpage structure
- `script.js` → Navigation button functionality
- `SoundX.png` → Website logo
- `BG.png` → Hero section image

---

## How to Run the Project

1. Download the project files
2. Keep all files inside the same folder
3. Open `index.html` in any browser

---

## About the Project

The main purpose of this project is to create a stylish product landing page for a headphone brand. The website contains sections for products, features, and contact details.

JavaScript is used to add smooth scrolling functionality to the navigation buttons.
